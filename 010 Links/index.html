Usando o target="_blank" e o rel="external" dizemos para o programa que o link que colocamos é externo e que deve abrir uma nova aba para ele.

Usando links internos

Esta é a primeira página, acesse a segunda pagina aqui

Para fazer isso, criamos um paragrafo e adicionamos o a (que é a tag que adiciona o link), ao criar um novo arquivo dentro da pasta em que esta a pagina um, colocamos o link do segundo arquivo na tag que esta no primeiro arquivo, assim quando o usuario clicar no link será enviado para a segunda pagina, para voltar para a primeira pagina, deverá ser feito o mesmo procedimento, mas com o link do primeiro arquivo.
